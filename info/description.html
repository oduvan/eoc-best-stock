You are given the current stock prices. You have to find out which stocks cost more.

Input: The dictionary where the market identifier code is a key and the value is a stock price.

Output: A string and the market identifier code.

Example:

{% if interpreter.slug == "js-node" %} 
bestStock({"CAC": 10.0, "ATX": 390.2, "WIG": 1.2}) == "ATX"
bestStock({"CAC": 91.1, "ATX": 1.01, "TASI": 120.9}) == "TASI"
{% else %} 
best_stock({"CAC": 10.0, "ATX": 390.2, "WIG": 1.2}) == "ATX"
best_stock({"CAC": 91.1, "ATX": 1.01, "TASI": 120.9}) == "TASI"
{% endif %}

Preconditions: All the prices are unique.
